格式
    返回值类型 访问器 函数名(参数列表) {函数体}

说明
    返回值类型
        一般类型:
            省略

        特殊类型:
            Null
                函数体没有返回值
                或 return;
                或 return null

            void
                实际上返回值为 null,
                但标明 void 后,
                该函数的返回值不可被使用

    访问器
        // 可选

        // 加上访问器后, 函数相当于变成了直接访问值的对象, 且使用时不用加括号

        get | 该对象只可取值, 且需要去掉 (参数列表)
        set | 该对象只可设置值, 且 (参数列表) 只能有一个参数, 返回值类型为 void, 且可省略

    函数体
        // 也可用语法糖形式, 直接返回表达式的值,
        // 若 返回值类型标明为 void, 也可正常使用, 但没有返回值
        
        int foo() => 表达式;
    
    匿名函数
        可以只保留 '() {}' 部分
