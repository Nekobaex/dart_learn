格式
    修饰符关键字 类型/类型关键字 对象名 = 调用构造函数/字面量/对象/类
    
修饰符关键字
    // 可选

    final   无法更改, 在声明时计算且赋值
    const   无法更改, 编译时, 存入常量
    late    只声明, 稍后赋值 (也可以直接赋值, 但语义不流畅)
        'late int foo' 等效于 'int? foo'

    
    // 若加了 final 或 const 修饰符关键字, 
    // 则类型(类型关键字 可省略)

类型
    Object, num, ...

类型关键字
    var     自动确定类型
    dynamic 任意类型

对象名
    即变量名

调用构造函数
    也可为字面值,
    或其他类和对象
    
字面量
    123, true, 'hello'

对象
    即其他对象名,
    其他变量

类
    即类型
